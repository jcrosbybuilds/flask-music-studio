# Flask Music Studio

A multi-page web application built with Python and Flask to demonstrate web routing, HTML templating, CSS styling, and dynamic content rendering.

## Technologies

- Python
- Flask
- HTML5
- CSS3
- Jinja2

## Features

- Multi-page website
- Shared layout using template inheritance
- Dynamic date and time
- Navigation bar
- External resource links
- Responsive project organization
